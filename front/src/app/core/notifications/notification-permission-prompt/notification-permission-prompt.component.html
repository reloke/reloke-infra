Activer les notifications ?

Autorisez Reloke à vous prévenir sur votre PC dès qu'un nouveau match est trouvé ou que vous avez un nouveau message.

Vous pourrez changer ce choix dans les réglages du navigateur.

Les notifications sont bloquées par votre navigateur.

Pour les activer, cliquez sur l'icône de cadenas à gauche de l'URL et autorisez les notifications, puis rafraîchissez la page.
